# Забудьте о постоянном переключении VPN:<br/> настраиваем исключения приложений в WireGuard

## Пошаговая инструкция:

1. Откройте приложение WireGuard
2. Нажмите на **название туннеля**

<div style="text-align: center; margin: 20px 0;">
  <img src="image/01.jpg" alt="step 1" width="30%" style="border: 2px solid #444444; border-radius: 8px; padding: 10px; background-color: #1e1e1e; box-shadow: 0 4px 8px rgba(0,0,0,0.5);" />
</div>

3. На следующем экране нажмите на иконку **карандаша** ️ в правом верхнем углу
<div style="text-align: center; margin: 20px 0;">
  <img src="image/02.jpg" alt="step 2" width="30%" style="border: 2px solid #444444; border-radius: 8px; padding: 10px; background-color: #1e1e1e; box-shadow: 0 4px 8px rgba(0,0,0,0.5);" />
</div>



4. В разделе **«Интерфейс»** найдите строку **«Все приложения»**
<div style="text-align: center; margin: 20px 0;">
  <img src="image/03.png" alt="step 3" width="30%" style="border: 2px solid #444444; border-radius: 8px; padding: 10px; background-color: #1e1e1e; box-shadow: 0 4px 8px rgba(0,0,0,0.5);" />
</div>

5. Нажмите на эту строку

6. Выберите режим:
   - **«Исключить для»** — выбранные приложения работают **в обход VPN**
   - **«Включить для»** — только выбранные приложения работают **через VPN**

<div style="text-align: center; margin: 20px 0;">
  <img src="image/04.png" alt="step 4" width="30%" style="border: 2px solid #444444; border-radius: 8px; padding: 10px; background-color: #1e1e1e; box-shadow: 0 4px 8px rgba(0,0,0,0.5);" />
</div>   

7. Отметьте галочками нужные приложения
<div style="text-align: center; margin: 20px 0;">
  <img src="image/05.jpg" alt="step 5" width="30%" style="border: 2px solid #444444; border-radius: 8px; padding: 10px; background-color: #1e1e1e; box-shadow: 0 4px 8px rgba(0,0,0,0.5);" />
</div> 

8. Нажмите на строку под окном выбора приложений **«Исключить ХХ прил...»** (или **«Включить ХХ прил...»**) для подтверждения

9. **СОХРАНИТЕ настройки:**
   - Нажмите иконку **дискеты** 💾 в правом верхнем углу

<div style="text-align: center; margin: 20px 0;">
  <img src="image/06.jpg" alt="step 6" width="30%" style="border: 2px solid #444444; border-radius: 8px; padding: 10px; background-color: #1e1e1e; box-shadow: 0 4px 8px rgba(0,0,0,0.5);" />
</div> 

   - Нажмите стрелку **«Назад»**
